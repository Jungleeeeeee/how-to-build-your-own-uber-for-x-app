# Система вызова экстренных служб в реальном времени (Uber-for-X)

Информационная система для мгновенной фиксации экстренных вызовов от гражданских лиц и их оперативного распределения между ближайшими дежурными сотрудниками служб реагирования в режиме реального времени. Проект модернизирован и развернут в рамках производственной практики.

## 🛠️ Стек технологий
* **Frontend:** HTML5, CSS3, JavaScript (ES6), Leaflet.js (автономные интерактивные карты OpenStreetMap)
* **Backend:** Node.js, Express.js, Socket.io (протокол WebSockets для обмена данными в реальном времени)
* **Database:** MongoDB (используется встроенный сервер `mongodb-memory-server` для полной автономности проекта)

## 📦 Модернизация и личный вклад
В ходе практики проект был полностью переработан для стабильной работы в локальном изолированном окружении:
1. Интегрирована встроенная база данных `mongodb-memory-server` с автоматическим парсингом исходных JSON-массивов (`cops.json`, `crime-data.json`) при старте приложения. Сторонние утилиты импорта больше не требуются.
2. Неработающая из-за блокировок CDN картографическая система Mapbox API полностью заменена на свободные карты Leaflet.js + OpenStreetMap.
3. Внешние фронтенд-зависимости (Axios, Leaflet) перенесены на локальное серверное хранение и статическую раздачу через Express для обеспечения автономности.
4. Исправлены уязвимости типизации Mongoose ORM и оптимизирована WebSocket-маршрутизация ответов исполнителей.

## 🚀 Инструкция по локальному запуску
1. Убедитесь, что на компьютере установлен [Node.js](https://nodejs.org/) (версии 16+).
2. Скачайте проект и откройте терминал в его корневой папке.
3. Установите все локальные зависимости:
   ```bash
   npm install
   ```
4. Запустите сервер приложения:
   ```bash
   node app.js
   ```
5. Откройте демонстрационные страницы в браузере бок о бок для проверки сценария:
   * **Панель пострадавшего гражданина:** [http://localhost:8000/civilian.html?userId=Vadim](http://localhost:8000/civilian.html?userId=Vadim)
   * **Панель дежурного сотрудника полиции:** [http://localhost:8000/cop.html?userId=01](http://localhost:8000/cop.html?userId=01)
